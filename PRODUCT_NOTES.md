# Opening Fit Product Notes

## Core value

Opening Fit analyses a user's Chess.com or Lichess games and recommends openings that match how they actually play.

## Free version

The free version should include:

- Chess.com import
- Lichess import
- Recent games analysis
- Opening win rates
- Style profile
- Keep / Improve / Avoid
- Basic opening recommendations
- Game replay
- Feedback form

## Premium ideas

Premium could include:

- 12-month analysis
- Deeper opening statistics
- Repertoire builder
- Stockfish-assisted opening review
- Mistake patterns by opening
- Exportable training plan
- Personalised opening course
- Saved account history
- Advanced comparison between platforms

## Launch goal

The first public version should feel useful even without premium.

A user should be able to:

1. Enter a username
2. Import their games
3. See what openings they play
4. See what openings work best
5. Get simple recommendations
6. Send feedback
