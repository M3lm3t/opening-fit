# Opening Fit Launch Checklist

## Must-have before public launch

- [ ] Chess.com import works
- [ ] Lichess import works
- [ ] Username not found shows a friendly error
- [ ] No games found shows a friendly error
- [ ] Backend offline shows a friendly error
- [ ] Last imported profile is saved after refresh
- [ ] Unknown openings are hidden by default
- [ ] Opening recommendations are easy to understand
- [ ] Keep / Improve / Avoid section is clear
- [ ] Game replay works
- [ ] Feedback form works
- [ ] Mobile layout looks good
- [ ] Burger menu and close button are centred
- [ ] Live frontend uses correct backend URL
- [ ] GitHub is updated
- [ ] Live site is updated

## Nice-to-have before launch

- [ ] Premium preview cards
- [ ] About Opening Fit section
- [ ] Simple analytics tracking
- [ ] Refresh analysis button
- [ ] Last analysed date shown
- [ ] Better loading messages
